My Goals

  • Master React.js and Next.js

    Mastering React.js will allow me to build dynamic and high-performance user interfaces, while Next.js will enable me to create robust, server-rendered applications.

  • Contribute to open source projects

    Contributing to open source projects will help me improve my coding skills, collaborate with other developers, and give back to the community.

  • Build a personal portfolio website

    A personal portfolio website will showcase my skills, projects, and professional journey. It's basically an online resume and helps in building a personal brand that potential employers and clients can easily access.

  • Learn React Native

    Learning React Native will allow me to build cross-platform mobile applications using JavaScript and React. This skill will expand my ability to develop for mobile platforms, broadening my development capabilities

  • Learn backend development with Node.js

    Node.js is a powerful backend technology that enables building scalable and efficient server-side applications. Learning Node.js will make me a more versatile developer, capable of working on both frontend and backend tasks.

  • Get certified in cloud technologies (AWS, Azure)

    Cloud computing skills are in high demand. Getting certified in AWS and Azure will validate my expertise in managing cloud infrastructure and services, enhancing my ability to work on cloud-based projects.
